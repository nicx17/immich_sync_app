Immich Desktop Client for Linux

Mimick

Keep your photo and video folders continuously synced to Immich with a clean GTK4 desktop app, tray controls, and resilient background uploads.

Mimick watches Linux folders in the background, uploads media to Immich, supports per-folder album rules, retries failed uploads, and gives you tray controls for pause, resume, sync now, and diagnostics.

Install

Method 1: Direct flatpakref

flatpak install --user https://nicx17.github.io/mimick/mimick.flatpakref

Method 2: Add repo + install

flatpak remote-add --user --if-not-exists mimick-repo https://nicx17.github.io/mimick/mimick.flatpakrepo
flatpak install --user mimick-repo io.github.nicx17.mimick

Use this option if you want the repository configured explicitly.

Note: If you receive a "Required runtime not found" error, ensure you have the Flathub remote configured.
